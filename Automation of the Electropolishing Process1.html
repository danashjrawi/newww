Transforming Manual Operations into Automated Precision

1. Manual Process Description (AS-IS)

The electropolishing process is currently performed manually on an existing laboratory electropolishing machine. The sequence of operations is as follows:

  1. The operator secures the wire in the clips on the vertical stand.
  2. The stand is manually lowered until the wire is fully immersed in the electropolishing solution.
  3. A stopwatch is used to measure the immersion time (approximately 40 seconds).
  4. After the set time, the operator manually raises the stand.
  5. The wire is removed, and the procedure is repeated for the next sample.

Challenge: This manual process relies entirely on the operator's judgment and reaction time. As a result, variations may occur in immersion depth and immersion time, which can affect process repeatability and consistency.

2. Automated Process Description (TO-BE)

In the automated system, the same process is executed automatically, while the electropolishing machine itself (bath, electrodes, and power supply) remains unchanged.

The goal: Automate the vertical motion and the timing, while keeping the same basic sequence.

Automated Process Steps:

  1. The operator mounts the wire in the clips and presses a START button.
  2. A motorized system automatically lowers the stand to a predefined immersion depth.
  3. When the lower position is reached, the system automatically starts a 40-second timer.
  4. After exactly 40s expires, the system automatically raises the stand.
  5. When the upper position is reached, the motor stops and the system signals process completion.

Key Point: Only the motion and timing are automated. The chemical and electrical aspects of electropolishing remain fully manual and unchanged.

Safety and Reliability Elements:

  • Upper and lower limit switches to define the exact end positions.
  • Emergency stop to immediately cut power to the motor.
  • Optional interlocks: bath temperature/level OK, hood door closed, etc.

This converts a manual timing-and-motion task into a repeatable automated cycle.

After defining the automated electropolishing sequence, several possible implementation approaches were examined.

All options implement the same automated motion-and-timing sequence, differing only in controller type, motor selection, and system complexity.

Description

The immersion time is controlled entirely by the microcontroller software. Position accuracy is achieved mechanically rather than through software position control.

  • Upper and lower limit switches ‚Äì define end positions and stop motion

Advantages

  • Lower cost compared to stepper motor‚Äìbased solutions

  • Position repeatability achieved through mechanical design rather than complex software

Disadvantages

  • Lower positioning resolution compared to stepper motor systems
  • Motion speed and smoothness depend on load, friction, and gearbox selection
  • Limited flexibility for variable or programmable immersion depths

6. Preliminary Mechanical, Software, and Control Design

Mechanical Design (Preliminary)

The DC motor is mechanically attached to the existing vertical stand using a motion-conversion mechanism such as a lead screw and nut or a belt and pulley system.

  • The motor produces rotational motion.
  • The mechanical transmission converts rotation into vertical linear motion.
  • The stand is guided by existing vertical rails, ensuring straight up-and-down movement.
  • There is no electrical connection between the motor and the stand.

This design allows the motor to replace the operator's manual force while maintaining the original mechanical structure.

Electrical Hardware Design (Preliminary)

The electrical system includes:

  • Microcontroller (Arduino or similar)
  • DC motor
  • Motor driver (H-bridge or relay module)
  • DC power supply
  • Upper and lower limit switches
  • Start, Reset, and Emergency Stop buttons
  • Status indicator (LED or buzzer)

The automation electronics are electrically isolated from the electropolishing power supply to ensure safety.

Software and Control Logic (Preliminary)

The control software executes the following sequence:

  1. Wait for Start command
  2. Activate motor to lower stand
  3. Stop motor when lower limit switch is activated
  4. Start 40-second immersion timer
  5. Activate motor to raise stand after timer completion
  6. Stop motor when upper limit switch is activated
  7. Signal end of cycle and return to idle state

The software controls motion and timing only. It does not control motor force or electropolishing parameters.
